**Short reflection answers**

Why is this page called dynamic?

This page is called dynamic because its content changes after the user clicks a button. The weather data is loaded from an API and shown on the page without manually writing the values in the HTML.

What does the API give us?

The API gives us real weather data in JSON format. It provides values such as temperature, wind speed, and time for the selected city.

Why is JSON useful here?

JSON is useful because it stores data in a clear structured format that JavaScript can easily read. It lets us access fields like data.current.temperature_2m and use them directly in the page.

Why is it better to create one reusable function for all cities?

A reusable function avoids repeating the same code many times. It makes the program cleaner, easier to maintain, and simpler to update if we want to add more cities later.
