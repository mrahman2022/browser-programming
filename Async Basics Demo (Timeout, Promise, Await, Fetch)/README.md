Why do we use async/await?
We use async/await to make asynchronous code easier to read and write. It helps the code look more like normal step-by-step code while still waiting for asynchronous tasks.

Why do we use try/catch with fetch?
We use try/catch with fetch to handle errors safely. It helps prevent the program from breaking when there is a network problem or another unexpected issue.

Why do we check response.ok?
We check response.ok to make sure the HTTP request was successful before using the returned data. Without this check, the program might continue even when the server returns an error status like 404.
